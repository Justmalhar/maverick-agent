# Maverick Agent

A self-hosted AI agent harness — inspired by Hermes Agent, built for the Maverick ecosystem.

## Overview

Maverick Agent is a lightweight, extensible agent runtime that connects LLMs to tools, skills, and external systems. Designed as the agentic backbone for the Maverick ecosystem — a drop-in brain that powers CLI tools, desktop apps, and voice interfaces alike.

## Quick Start

```bash
# Install dependencies
bun install

# Run the agent
bun run agent
```
