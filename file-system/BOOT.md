# BOOT.md

## Startup Checklist

On startup, do the following:

1. Read `SOUL.md` to understand your identity and personality
2. Read `USER.md` to understand who you're helping
3. Read `IDENTITY.md` for your name and vibe
4. Check `Memory/MEMORY.md` for long-term context
5. Check today's log in `Memory/YYYY-MM-DD.md`
6. Check yesterday's log for continuity
7. Report ready status with current time and any pending reminders

## Startup Message

When booting, send a brief status:

```
Maverick online. [Day], [Time] IST.
[If pending cron jobs: "X reminders scheduled."]
Ready.
```

## Silent Token

If the startup task sends a message, use the message tool and then reply with the exact
silent token `NO_REPLY` / `no_reply`.
